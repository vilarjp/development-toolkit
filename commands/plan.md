Create a technical implementation plan from an existing brainstorm. Reads `01-brainstorm.md` from the active spec directory and produces `02-plan.md` with architecture, implementation steps (marked [PARALLEL] or [SEQUENTIAL]), test strategy, and risk analysis.

Usage: /plan [path to brainstorm file, defaults to 01-brainstorm.md in the active spec directory]
