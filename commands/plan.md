Create a technical implementation plan from an existing brainstorm. Reads `docs/spec/01-brainstorm.md` and produces `docs/spec/02-plan.md` with architecture, implementation steps (marked [PARALLEL] or [SEQUENTIAL]), test strategy, and risk analysis.

Usage: /plan [path to brainstorm file, defaults to docs/spec/01-brainstorm.md]
